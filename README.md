# Simple Presence

This is a simple, single-page website that displays a logo in the center of the page with a gradient background.

## Files

*   `index.html`: The main HTML file.
*   `style.css`: The CSS file for styling the page.
*   `trashpanda.png`: The logo image.

## Running Locally

To view the website locally, simply open the `index.html` file in a web browser.

## Deployment

This project includes Terraform configuration for deploying to AWS using S3 and CloudFront.

See the [terraform/README.md](terraform/README.md) for detailed deployment instructions.
