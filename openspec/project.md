# Simple Presence Project

## Overview
Simple Presence is a single-page static website that displays a logo (trashpanda.png) centered on a page with a gradient background. The site uses Tailwind CSS for layout and custom CSS for styling.

## Current Architecture
- **Type**: Static HTML/CSS website
- **Files**:
  - `index.html`: Main HTML file
  - `style.css`: Custom styling with gradient background
  - `trashpanda.png`: Logo image
- **Dependencies**: Tailwind CSS (via CDN)
- **Hosting**: Currently local only

## Technology Stack
- HTML5
- CSS3
- Tailwind CSS (CDN)

## Deployment
Currently, the site is deployed by opening `index.html` directly in a web browser.
