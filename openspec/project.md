# Simple Presence Project

## Overview

Static single-page website displaying a logo centered on a gradient background.

## Current State

- Static files: `index.html`, `style.css`, `trashpanda.png`
- Local development via `python3 -m http.server 4173` or `npx serve . --listen 4173`
- No deployment infrastructure currently configured

## Architecture

- Frontend: Single HTML page with CSS styling
- Assets: Static image file (trashpanda.png)
- No backend or server-side processing required
